Lista de Definiciòn

BIOS
Basic Input Outup System

El sistema básico de entrada/salida (BIOS) es la parte de un sistema operativo que
vincula los dispositivos de hardware específicos con el software. Obtiene los búferes
necesarios para enviar información desde un programa al hardware/escritorio que recibe
la información.

RAM
Random Access Memory

La RAM (memoria de acceso aleatorio) es la memoria a corto plazo de un ordenador,
donde se almacenan los datos que el procesador está utilizando en ese momento.

HTML5
Lenguaje de Marcado de Hipertexto ver. 5

El lenguaje de marcas de hipertexto (más conocido como HTML) es un lenguaje de
formato de documentos que se emplea en la creación de páginas web y aplicaciones online.
El código HTML permite a su autor definir la estructura de una página web y el aspecto
que mostrará en un navegador.

CSS 3
Cascading Style Sheets ver. 3

Sus siglas corresponden a “Cascading Style Sheets”, que tiene el siguiente significado: Cascading, que
significa que los estilos que aplicamos a los elementos de una página web se propagan a los elementos
que contiene, se propagan en cascada.
